KEY CLAIMS:
- Identity increases subjective value of identity-congruent behaviors
- Self-regulation operates via dynamic value integration across choice attributes
- Goal-directed behaviors that are identity-relevant have greater subjective value
- Neural overlap in vmPFC between identity processing and valuation
- Making target behaviors more self-relevant improves self-regulation
- Challenges dual-process (hot vs. cold) framing of self-control
- Identity as strong, enduring source of value in self-regulation outcomes
